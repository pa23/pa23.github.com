  6. Строка ввода в диалоге ввода значений теперь всегда в фокусе, что облегчает работу по модификации данных таблиц.

22.05.2010 - Релиз r49_10.30

Логика расчета теперь основывается на исходных данных (Если в расчете необходимо использовать замеренные значения расхода воздуха, то массив Gair должен содержать результаты замеров, а dPn - нули. Для остальных случаев - аналогично).

  5. В связи с изменением файла исходных данных (добавлена колонка с параметром rd), пересчет ранее сделанных расчетов возможен только при добавлении колонки rd в конце файла с помощью любого табличного или текстового редактора. Qr49 также можно для этого использовать (открыть файл исходных данных в редакторе отчетов ))).
