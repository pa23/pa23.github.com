  2. Удалены параметры Расчет Ne нетто (NenCalc), Значения Gв (GairVals) и Расчет NOx (NOxCalс). Логика расчета теперь основывается на исходных данных (Если в расчете необходимо использовать замеренные значения расхода воздуха, то массив Gair должен содержать результаты замеров, а dPn - нули. Для остальных случаев - аналогично).
  3. Добавлена проверка корректности исходных данных при расчете частот вращения коленчатого вала двигателя A, B, C.

  5. В связи с изменением файла исходных данных (добавлена колонка с параметром rd), пересчет ранее сделанных расчетов возможен только при добавлении колонки rd в конце файла с помощью любого табличного или текстового редактора. Qr49 также можно для этого использовать (открыть файл исходных данных в редакторе отчетов ))).
